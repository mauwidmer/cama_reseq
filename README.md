# cama_reseq

This project was conducted as a lab rotation for the CBB master's degree.

## Overview

This repository contains scripts and configuration files for data processing and quality control of C. amara collected by the Shimizu Lab.

## Directory Structure

* `scripts/` – analysis and QC scripts
* `results/` – generated outputs (not tracked by Git)
